🚗 Sistema de Locadora de Carros
📖 Sobre o Projeto

Este projeto consiste em um sistema de gerenciamento de uma locadora de carros, desenvolvido em linguagem C como atividade acadêmica do 2º termo do curso de Ciência da Computação.

O sistema roda em modo console e permite realizar o gerenciamento de clientes, carros e locações, utilizando arquivos binários (.dat) para persistência dos dados.

O objetivo principal foi aplicar conceitos como manipulação de arquivos, estruturas de dados, busca, ordenação e organização de sistemas com menus.

⚙️ Funcionalidades

O sistema permite:

👤 Clientes
Cadastrar clientes
Alterar dados de clientes
Excluir clientes
Listar clientes cadastrados
🚗 Carros
Cadastrar carros
Alterar dados de carros
Excluir carros
Listar carros cadastrados
📄 Locações
Registrar locações de veículos
Alterar locações
Excluir locações
Listar locações realizadas
📊 Relatórios

O sistema também gera relatórios como:

Clientes com mais de 3 locações
Carros por categoria
Locações com valor acima de um limite
Busca de locações por CPF
Relatório completo com clientes, carros e locações
